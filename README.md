# نظام إدارة الإعمار الاحترافي (Professional Construction Management System)

هذا المشروع هو نسخة مطورة وشاملة لنظام إدارة مشاريع البناء والإعمار.

## الميزات الجديدة والمطورة:
1. **هيكلة احترافية**: تم فصل المنطق البرمجي عن واجهات العرض (MVC-like) لتسهيل الصيانة والتطوير.
2. **تصميم عصري**: واجهة مستخدم محسنة بالكامل باستخدام Bootstrap 5 و CSS مخصص، متجاوبة مع كافة الشاشات والجوال.
3. **الأمان**: تم تنظيف الكود من الثغرات، وفصل بيانات قاعدة البيانات، واستخدام PDO بشكل آمن.
4. **سجل النشاطات (Log)**: نظام لمراقبة كافة العمليات التي تتم في النظام ومعرفة من قام بكل إجراء.
5. **إدارة المرفقات**: دعم رفع ملفات متعددة (صور وفواتير PDF) لكل مادة يتم شراؤها.
6. **نظام البحث**: خاصية البحث الفوري في جداول المواد والعمال لتوفير الوقت.
7. **تنبيهات الميزانية**: تنبيه ذكي عند تجاوز المصروفات لنسبة معينة من الميزانية المرصودة.
8. **إدارة العمال والموردين**: إضافة حقول جديدة للتواصل (الهاتف والعنوان) مع إمكانية التواصل المباشر عبر واتساب.

## تعليمات التثبيت:
1. قم برفع كافة الملفات إلى الخادم الخاص بك.
2. تأكد من إعداد قاعدة البيانات في ملف `config/db.php`.
3. قم بتشغيل الاستعلامات الموجودة في ملف `database_updates.sql` لتحديث هيكل الجداول.
4. تأكد من أن مجلد `uploads` لديه صلاحيات الكتابة.

## هيكل المجلدات:
- `assets/`: ملفات CSS و JavaScript.
- `config/`: إعدادات قاعدة البيانات.
- `includes/`: المنطق البرمجي والوظائف المساعدة.
- `templates/`: واجهات العرض (HTML).
- `uploads/`: الملفات والمرفقات المرفوعة.
